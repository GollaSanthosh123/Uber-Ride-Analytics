# Uber Ride Analytics Dashboard

## Project Overview

This project is an interactive **Uber Ride Analytics Dashboard**
developed using **Microsoft Power BI**. The dashboard analyzes
ride-booking data to provide insights into booking trends, vehicle
performance, revenue, cancellations, and customer/driver ratings.

The report is organized into multiple interactive pages with date
filters and navigation buttons, allowing users to explore operational
and business performance from different perspectives.

## Dashboard Pages

### 1. Home Page

The landing page provides navigation to the main analytical sections of
the report.

### 2. Overall Analysis

This page gives a high-level view of ride-booking performance.

Key analysis includes: - Booking trends over time - Distribution of
booking status - Date-based filtering - Overall booking activity
monitoring

### 3. Vehicle Type Analysis

This page compares ride performance across different vehicle categories.

Key analysis includes: - Booking value by vehicle type - Ride distance
by vehicle type - Comparison of vehicle-category performance -
Date-based filtering for dynamic analysis

### 4. Revenue Analysis

This page focuses on revenue and customer contribution.

Key analysis includes: - Booking value by payment method - Ride-distance
trends over time - Customer-wise booking value - Identification of
high-value customers - Date-based revenue filtering

### 5. Cancellation Analysis

This page analyzes ride cancellations from both customer and driver
perspectives.

Key analysis includes: - Customer cancellation reasons - Driver
cancellation reasons - Cancelled rides by customers - Cancelled rides by
drivers - Cancellation percentage - Date-based cancellation analysis

### 6. Ratings Analysis

This page evaluates service quality using customer and driver ratings.

Key analysis includes: - Customer ratings across vehicle categories -
Driver ratings across vehicle categories - Comparison of customer
satisfaction and driver performance

## Tools and Technologies

-   **Microsoft Power BI** -- Dashboard development and data
    visualization
-   **Power Query** -- Data cleaning and transformation
-   **DAX** -- Calculated measures and analytical logic
-   **Data Modeling** -- Structuring data for interactive analysis

## Key Business Insights

The dashboard is designed to help stakeholders:

-   Monitor booking volume and ride trends over time
-   Understand the contribution of different vehicle categories
-   Analyze revenue by payment method and customer
-   Identify major reasons behind ride cancellations
-   Compare customer and driver rating patterns
-   Support data-driven decisions for improving ride operations and
    customer experience

## How to Use the Project

1.  Download or clone this repository.
2.  Install **Microsoft Power BI Desktop**.
3.  Open the `Uber Ride Analytics.pbix` file.
4.  Use the navigation buttons to move between dashboard pages.
5.  Apply the date slicers to explore performance for selected periods.
6.  Interact with charts and visuals to analyze specific segments.
